《Ring V5：裂变与聚合》—— 动效设计文档（骨架版）
■ 0. 动画总长度

使用 normalized time：
t ∈ [0.0, 1.0]
实际时长可后期设定（建议 10–15 秒）

■ 1. Horizon 主路径

ring 的中心点沿着一个半径极大的弧线从左向右运动：

左边低

右边高

整体轻微抛物线/地平线效果

方向是“向前＋向右”

数值不固定，后期可调。

■ 2. 初始阶段（Phase A：t = 0.0–0.3）

一个 ring（n = 2）

轻微呼吸

沿 Horizon 缓慢向前

准备进入变化区

■ 3. 预震阶段（Phase B：t = 0.3–0.5）

仍然只有一个 ring

n 在 1.8～2.2 之间轻波动

提示“不稳定”

■ 4. 裂变阶段（1→3）（Phase C：t = 0.5–0.7）

从一个 ring 分裂成三个 Lamé 曲线：

三个形态（n 值）：

Soft（柔）：n ≈ 1.2

Core（原型）：n = 2

Hard（硬）：n ≈ 4

空间展开方式：

三个 ring 的中心均从 C_base（主路径） 分离一点点：

C₁ → 往“右上 60°”偏一点

C₂ → 跟着主路径向前，不偏不倚

C₃ → 往“右下 60°”偏一点

偏移量小，不炸开，只是轻微散开。

原始 ring 同时淡出。

■ 5. 多态阶段（Phase D：t = 0.7–0.85）

三个形态同时向前移动：

大方向一致

Soft/Hard 在上下缓慢起伏

各自的 n 做轻微的局部振荡（不脱离各自区间）

视觉上是“内部多形态并存”的效果。

■ 6. 聚合阶段（3→1）（Phase E：t = 0.85–1.0）

你选了方案 A：

✔ 三条曲线逐渐靠拢 + 重叠 + 叠加变厚

C₁、C₂、C₃ 的中心重新合拢到一起

各自的 n 同步往 2 靠拢

三条 ring 越来越重叠，亮度/厚度提高

最后形成一个比原来更稳的 updated ring（n≈2）
