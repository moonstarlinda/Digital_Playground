# 🎨 Digital Playground  
**A tiny space where wandering particles find their gentle shape.**  
灵感粒子的小小游乐场 —— 它们在混乱中相遇，最终组成温柔、可见的形状。

---

## 🌟 About this project  
**Digital Playground** 是一个以 “生成式视觉 + 轻交互” 为核心的小型实验场域。  

它记录了我在学习代码、美学与数字创作过程中的作品：  
- 从最初的单环粒子  
- 到渐渐复杂的双环轨道  
- 再到未来可能出现的更多形式  

它既是技术练习，也是灵感漂浮的地方。
